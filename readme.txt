== Description ==

Pixie Dust creates a redirect to the page a site visitor was on before going to the login page; 
If viewing an item, gallery, or post, after logging in the visitor will be directed back to that page.

If redirect is null or login page, redirects to homepage.

Minimal code, no bloat.


Use shortcode [pixie_login_btn] to place a link on to your pages/posts.  
Style class 'pixiedust_ecs_log-in' is associated with the link.




== Installation ==



Upload the Pixie Dust plugin to your blog, Activate it.

Use shortcode to place link on to pages/posts.
